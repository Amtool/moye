控件选项配置

Properties
Name Type Attributes Description
min number

范围选择输入控件最小值，默认0
max number

范围选择输入控件最大值，默认100
step number

滑块滑动选择增加/减小的步长，大于0的数，默认值为1
value number | Object

滑块选择的值或区间， 对于非区间情况下默认值为最小值和最大值平均值，区间情况下，默认为[min, max]
range boolean <optional>

是否提供范围选择即提供两个滑块，默认一个
scale Object <optional>

显示范围选择的刻度信息，默认不显示

Properties
Name Type Attributes Description
stick boolean | Array.<number>=

是否显示刻度线，默认不显示 也可以指定要显示刻度线对应刻度值列表，若值为true，默认刻度线跟要显示刻度值列表一样
values Array

要显示的刻度值列表 可以这样：[10, 20, 30]，[10, 20, {value: 30, format: '$!{v}以上'}] value里指定的 format 优先级高于 scale.format
format string <optional>

刻度值格式化模板，可选, 默认原值显示， e.g., '$!{v}'
valueLabel Object | boolean

滑块选择的值的标签信息，可选，默认不显示

Properties
Name Type Attributes Description
dragShow boolean <optional>

是否拖拽时候动态显示滑块当前值， 默认 false
format string <optional>

值标签格式化模板，可选, 默认 原值显示，e.g., '$!{v}'
formater function <optional>

自定义的值格式化方法，表单提交值使用该格式化 后的值，默认区间值提交值格式为 12-34
scaleStickTpl string <optional>

刻度线模板
scaleValueTpl string <optional>

刻度值模板
responsive boolean <optional>

是否支持响应式，默认，false
