Module: Cookie

Cookie 模块

Source:
  • Cookie.js, line 43
Example
var cookie = new Cookie('foo', {
    duration: 1, // 1 天
    domain: 'foo.com',
     path: '/'
 });
cookie.set('bar');
cookie.get(); // 'bar'

或者使用静态方法：

Cookie.set('foo', 'bar');
Cookie.get('foo');   // 'bar'
