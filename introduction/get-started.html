设定 / Setup

设定 amd 配置

moye 是基于 amd 模式开发的，因此在正式使用前，需要对 amd 模块加载器进行配置。

NOTICE: moye 并不支持在 global 变量模式或者 cjs 模式下使用。建议使用 amd 来构建你的项目

设定 less 编译路径

moye的样式源码是使用 less 进行编写的。
我们使用了 est 来简化我们在开发 less 时的工作。est 这是一个非常强大的 less mixin 库。
因此，在你的 less 编译参数中，需要将 est 的路径加入到 paths中。

一将来讲 est 会在 bower install 一起安装到你的项目，位于 moye 同一级的目录。
如果没有通过 bower 安装，那么可以通过这条命令来安装它
